# LEODAN WEB

Sitio web oficial de LEODAN, DJ y productor independiente de House / Tech House.

## Sobre mí

Mi nombre es Leonardo Bueno y mi nombre artístico es LEODAN. Soy un artista independiente nacido en Trinidad, Flores, Uruguay, con una pasión profunda por la música electrónica.

Hace más de 5 años comencé este camino musical, formándome como DJ en Groove Beats DJ School, perfeccionándome luego en la academia Soulmate de Montevideo y complementando mi desarrollo con estudios de Ingeniería Musical online desde España.

Mi sonido se mueve entre el House y el Tech House, con una identidad oscura, elegante, energética y emocional. Busco crear música que conecte desde lo profundo, combinando groove, atmósferas envolventes y una visión artística auténtica.

## Proyecto

Este repositorio contiene la base de mi futura página web artística. La idea es construir una landing page profesional con:

- Biografía artística
- Música y lanzamientos
- Press kit
- Redes sociales
- Contacto profesional
- Identidad visual oscura, minimalista y con tonos verdes

## Objetivo

Crear una web profesional para centralizar mi identidad como DJ y productor, mostrar mi música, facilitar contactos profesionales y construir una presencia digital sólida.
