# Demo Follow-up

Sistema simple para seguimiento de demos enviados a sellos.

## Datos a registrar

- Sello.
- Contacto.
- Email o canal.
- Track enviado.
- Link privado usado.
- Fecha de envio.
- Fecha de follow-up.
- Respuesta.
- Estado.

## Estados

- Pendiente de envio.
- Enviado.
- Follow-up 1.
- Respondio.
- Interesado.
- No interesado.
- Sin respuesta.

## Cadencia sugerida

- Dia 0: envio inicial.
- Dia 7 a 10: primer follow-up.
- Dia 21: cierre suave si no hubo respuesta.

## Follow-up breve

Hola [NOMBRE],

Solo queria hacer un breve seguimiento del demo de LEODAN que envie: "[NOMBRE_TRACK]".

Link privado: [LINK_PRIVADO_CONFIRMADO]

Gracias por escuchar.

LEODAN

## Reglas

- No insistir de forma agresiva.
- No enviar links no confirmados.
- Mantener tono profesional y directo.
- Registrar cada contacto para no duplicar envios.
