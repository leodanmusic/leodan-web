# Release Checklist

Checklist operativo para cada lanzamiento de LEODAN.

## Antes del anuncio

- Confirmar nombre exacto del track.
- Confirmar portada con solo el nombre del tema.
- Confirmar fecha si aplica.
- Confirmar links oficiales o marcar como pendientes.
- Preparar narrativa central.
- Preparar bio corta del lanzamiento.
- Preparar landing mobile-first.
- Preparar captura de emails.

## Contenido

- Reel teaser.
- Post de portada.
- Stories con CTA.
- Short de YouTube.
- Visualizer si aplica.
- Copy para Spotify.
- Copy para SoundCloud si hay link confirmado.

## Web

- Actualizar lanzamiento destacado.
- Revisar CTA principal.
- Revisar links.
- Revisar mobile.
- Revisar formulario de email.
- Revisar contacto.

## Sellos y press

- Preparar pagina de sello si aplica.
- Preparar demo email.
- Preparar press kit.
- Confirmar link privado si corresponde.

## Despues del lanzamiento

- Publicar link oficial.
- Compartir clips durante 2 semanas.
- Pedir guardados y escucha completa con CTAs naturales.
- Revisar rendimiento.
- Registrar aprendizajes para el proximo lanzamiento.
