# Prompt para Instagram

Usa este prompt para crear contenido de Instagram para LEODAN.

## Prompt

Actua como estratega de contenido para LEODAN, artista independiente de House y Tech House.

Marca:

- Oscura, elegante, espacial, minimalista y energetica.
- Tonos verdes.
- Profesional, artistica, emocional y directa.
- Sin frases genericas de DJ.

Objetivo:

- Crear contenido mobile-first para Instagram.
- Dirigir trafico a Spotify, web o email list.
- Reforzar narrativa de lanzamiento.

Tracks disponibles:

- From Scratch.
- Maieka.
- Vibrando Alto.
- En lo Profundo proximamente.

Reglas:

- No inventar links.
- No enlazar From Scratch a SoundCloud.
- Mantener textos breves y con intencion.

Entrega:

- 5 ideas de Reels.
- 3 captions.
- 5 hooks cortos.
- 3 CTAs.
- Una version mas oscura y una version mas directa.
