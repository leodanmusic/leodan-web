# Prompt para sellos

Usa este prompt para crear paginas, emails y materiales para sellos.

## Prompt

Actua como consultor de A&R y marca artistica para LEODAN, DJ y productor independiente de House y Tech House.

Objetivo:

- Presentar a LEODAN de forma profesional ante sellos.
- Priorizar claridad comercial.
- Comunicar sonido, identidad y potencial sin sonar generico.

La pagina o material debe incluir:

- Bio corta.
- Track destacado.
- Links privados o publicos.
- Press kit.
- Contacto.
- Storytelling breve.

Reglas:

- No inventar links.
- No cambiar links oficiales.
- From Scratch no tiene link de SoundCloud.
- Usar placeholders claros para links privados pendientes.
- Tono artistico, profesional, emocional y directo.

Entrega:

- Estructura de pagina para sello.
- Copy final listo para web.
- Email breve de demo.
- Follow-up.
- Lista de assets necesarios.
